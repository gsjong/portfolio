# portfolio

PP 폴더는 안드로이드 스튜디오를 이용하여 JAVA로 간단한 모바일 게임을 제작한 코드가 담긴 폴더입니다.
소스코드의 경로는 다음과 같습니다.
PP\app\src\main\java\edu\skku\cs\pp

소스코드의 경로는 다음과 같습니다.
practice\src\TP

project 폴더는 html과 css, Typescript를 이용하여 flee market 웹 페이지를 구현한 코드가 담긴 폴더입니다.
